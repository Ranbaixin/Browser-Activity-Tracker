# 前端展示层 (/src)

本文件夹包含了插件的 Web 介绍页面代码，采用 **React + Tailwind CSS** 构建。

## 核心组件说明

*   **`App.tsx`**: 
    *   主入口组件。
    *   定义了插件的特性介绍卡片（Feature Cards）。
    *   展示了插件的设计理念和安装引导。
*   **`components/`**:
    *   包含可复用的 UI 元素，如 `FeatureCard`。
*   **`index.css`**:
    *   全局样式文件，集成了 Tailwind CSS 的基础配置。

## 部署说明

该部分代码在构建时会生成静态文件并存放在 `/dist` 目录中。在插件的实际运行中，主要起到展示和文档说明的作用。
